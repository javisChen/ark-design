## kt-cloud这是什么？
这是我个人理想中的研发架构建设模型，整体会分为基础通用组件、基础服务、通用业务服务、监控、CI/CD、统一后台操作界面等。

## 目标
1. 提升研发效能
2. 标准化研发流程
3. 快速搭建，稳定支撑业务稳定运行
